This project is a modification of a bubble chart project from the Udemy course Mastering Data Visualization with D3.js, Project 2 - Gapminder Clone. It itself was inspired by a Gapminder visualization. The data and information for this project was inspired by Our World in Data.

  • Mastering data visualization in D3.js by Adam Janes
  • Gapminder site
  • Our World in Data site

This project was adapted from many different line charts that are developed using D3. The inspiration for this project is the many women who struggle with infertility and the hardships of trying to start or grow their families.

This project was partly inspired by a chloropleth map from the Udemy course Mastering Data Visualization with D3.js. Special thanks to Eric Celeste for his GeoJSON file of the US, and is adapted from the Basic US State Map - D3 by Michelle Chandra.
